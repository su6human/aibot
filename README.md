# 🚀 Digital Twin Career Engine

## 📖 Описание проекта
**Digital Twin Career Engine** — это персонализированный AI-дашборд, созданный специально для управления карьерным треком и обучения. Проект помогает визуализировать переход из сферы hospitality (бармен) в Tech-индустрию (Interactive Design, UI/UX, Game UI). 

Дашборд анализирует текущие хард- и софт-скиллы, предсказывает наиболее подходящие профессии на базе машинного обучения, а также включает в себя **"Live Coach"** — цифрового AI-двойника, который мотивирует, дает практические советы по изучению кода (JavaScript, Godot) и жестко "прожаривает" за прокрастинацию.

---

## 🏗️ Архитектура и Технологии (Tech Stack)

Проект разделен на две основные части: **Модель предсказания (Data/ML)** и **Пользовательский интерфейс (Frontend/Dashboard)**.

### 1. Frontend / UI (Дашборд)
- **Фреймворк:** `Streamlit` (Python). Выбран для быстрой разработки интерактивных веб-приложений и работы с данными.
- **Стилизация:** Кастомный CSS внутри Streamlit. Реализована темная тема с современным неоновым дизайном (акценты `#00ffcc` и `#b366ff`).
- **Визуализация данных:** `Plotly` (`plotly.graph_objects`). Используется для отрисовки интерактивного радар-чарта (Balance Wheel), наглядно сравнивающего Hard и Soft навыки.

### 2. AI Интеграция (Live Coach)
- **Провайдер:** `Groq API` — обеспечивает сверхбыструю генерацию текста.
- **LLM Модель:** `llama-3.3-70b-versatile`.
- **Логика:** Чат работает с сохранением контекста через `st.session_state` и использует **Streaming** (потоковый вывод текста) для создания ощущения живого общения. Системный промпт жестко задает персону коуча, знающего бэкграунд пользователя.

### 3. Backend / Data Processing (Модель навыков)
- **NLP & ML:** `scikit-learn` (`TfidfVectorizer`, `cosine_similarity`) в скрипте `career_model.py`. Модель векторизует текущие навыки пользователя и сравнивает их с требованиями рынка, рассчитывая процент совпадения (Match %).
- **Хранение данных:** Результаты работы модели экспортируются в `career_prediction.json`, откуда дашборд (`app.py`) подтягивает актуальные данные для отображения без необходимости каждый раз пересчитывать модель.

## 📂 Структура проекта
```text
Digital_Twin_Career/
│
├── app/
│   └── app.py                  # Главный файл Streamlit дашборда
│
├── career_advisor/
│   ├── career_model.py         # Скрипт ML-модели (сравнение навыков)
│   └── career_prediction.json  # JSON-база с предсказаниями и missing skills
│
└── README.md                   # Документация проекта
Основные модули (Вкладки)
🏠 Home: Главный экран с приветствием и фокусом на главную карьерную цель (Top Match).
📈 Career Prediction: Вывод топ-3 профессий в процентах и список навыков, которые необходимо подтянуть (Missing Skills).
⚖️ Balance Wheel: Радар-чарт, отображающий текущий баланс между техническими (Figma, Godot, HTML/CSS) и гибкими (Teamwork, Communication) навыками.
💬 Live Coach: Интерактивный чат с AI-ментором на базе Groq для обсуждения плана обучения и портфолио.
🔥 Roast My Stack: Юмористическая "агрессивная" кнопка мотивации в стиле строгого техлида.
🌳 RPG Tech Tree: (В разработке) Геймифицированное дерево прокачки навыков.
